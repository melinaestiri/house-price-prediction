# house-price-prediction
House Price Prediction with RandomForest

Description:
A Python project that predicts house prices using machine learning models. The dataset is preprocessed and a Random Forest Regressor is used for prediction.

Features:
Data preprocessing with Pandas and NumPy
Feature selection and engineering
Model training using Random Forest Regressor
Evaluation with RMSE, MAE, and RÂ²

How to run:

# Clone the repo
git clone https://github.com/melinaestiri/house-price-prediction.git

# Go to project folder
cd house-price-prediction

# Install required packages
pip install -r requirements.txt

# Run the main script
python main.py


Dependencies:
Python 3.x
Pandas
NumPy
scikit-learn
